## 项目结构

```
.
├── App.css (默认样式)
├── App.tsx (入口文件)
├── components
│   ├── Controller.tsx (控制输入)
│   └── RightClickMenu  (右键菜单)
│       ├── index.css
│       └── index.tsx
├── Context (全局上下文)
│   ├── ContainerContext.ts
│   ├── NewPointContext.ts
│   └── PointInfoContext.ts
├── d3_components (d3组件)
│   ├── canvas.ts (画布部分)
│   ├── point.ts (节点部分)
│   └── type.ts (公用类型)
├── index.css (默认样式)
├── index.tsx (入口文件)
├── logo.svg (logo)
├── mock (mock数据, 暂时不用)
│   └── data.ts
└── react-app-env.d.ts (不知道啥玩意儿,自动生成的)
```
